Essential Tools and Supplies for Gunpla

Basic Tools

  • Nippers or side cutters
  • Hobby knife
  • Sandpaper (various grits)
  • Tweezers
  • Cutting mat

Painting Supplies

  • Primer
  • Acrylic or enamel paints
  • Paintbrushes
  • Airbrush (for advanced builders)
  • Thinner and cleaner

Detailing Supplies

  • Panel line accent color
  • Gundam markers
  • Decal setter and softer
  • Topcoat

For more information on how to use these tools, check out our Building Techniques page.
